Description

Warnings System for PHP-Fusion. There are two types of counting warnings - number or percent. Moreover depending on the number of warnings can be set different blockades. By default, there are three - Forum, Shoutbox and Comments (without modifying files).

PointX Connection

Warnings System can be connected to infusion "PointX". This allows you to subtract points for added warning. The required version is 2.0.2 or later.

Update

Manual Update
1. Upload all files from the directory files on the server.
2. Refresh Website.
3. Update Completed.
Automatic update
1. Go to the Infusion management panel.
2. Click "Update".
3. Update Completed.

Automatic Update may not work correctly on some servers!
